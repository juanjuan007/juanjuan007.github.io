[TIOJ 1868](https://tioj.ck.tp.edu.tw/problems/1868)

<br>

**題敘**

有 $N$ 棟房子，每棟建築有高度 $h_{i}$。

一開始你在第 $K$ 棟，每次你可以往相鄰且高度不增的建築移動，

其中某些建築物上會有噴射裝置，若你到那幾棟則可以無視上述規則，走到任意一棟。

求最多可以走過多少不同建築。

$1 \leq N \leq 3 \times 10^{5}$ 、 $1 \leq K \leq N$ 、 $1 \leq h_{i} \leq 10^{6}$

<br>


**思路**


&lt;M一&gt;

將可從 $a$ 走到 $b$，視為存在一條有向邊 $a \rightarrow b$；有噴射裝置，代表對於每個節點都有一條邊。

至此，只要 SCC 縮點 + DAG DP 就可以解決問題。

特別注意建圖的方式，如果你對於每個噴射裝置，都再建一次是浪費時間的，最糟會到 $O(N^{2})$ ，但可以建一個虛點來處理所有的噴射裝置。


Time : $O(N)$

<br>

Code :

```cpp
#include<bits/stdc++.h>
using namespace std;

int N,K , t =1;
vector<vector<int>> g , SCC , dag;

vector<int>in , dfn , low , w , id , dp , vis;

stack<int>stk;

void init(){
    g.resize(N+1);
    in.resize(N+1);
    dfn.resize(N+1);
    low.resize(N+1);
    w.resize(N+1);
    id.resize(N+1);
    dp.resize(N+1);
    vis.resize(N+1);
    dag.resize(N);
}

void dfs(int v){
    dfn[v] = t++; low[v] = dfn[v];
    stk.push(v); in[v] = 1;
    for(int u : g[v]){
        if(dfn[u]==0){
            dfs(u);
            low[v] = min(low[u] , low[v]);
        }else if(in[u]){
            low[v] = min(dfn[u] , low[v]);
        }
    }

    if(dfn[v]==low[v]){
        int u;
        SCC.push_back({});
        do{
            u = stk.top(); stk.pop();
            in[u] = 0;
            SCC[SCC.size()-1].push_back(u);
            id[u] = SCC.size()-1;
        }while(u!=v);
    }
}

int dfs1(int v){
    if(vis[v]) return(dp[v]);

    dp[v] = w[v];
    vis[v] = 1;
    int x = 0;
    for(int u : dag[v]){
        x = max(dfs1(u) , x);
    }
    dp[v] += x;
    return(dp[v]);
}

int main(){
    ios::sync_with_stdio(0) , cin.tie(0);

    cin>>N>>K;
    K--;
    vector<int>h(N);

    for(int i=0;i<N;i++) cin>>h[i];
    
    init();

    for(int i=0;i<N;i++){
        if(i && h[i] >= h[i-1]) g[i].push_back(i-1);
        if(i+1 < n && h[i] >= h[i+1]) g[i].push_back(i+1);
    }

    char c;
    for(int i=0;i<N;i++){
        cin>>c;
        if(c=='T') g[i].push_back(n);
    }

    for(int i=0;i<N;i++) g[N].push_back(i);

    for(int i=0;i<=N;i++)
        if(dfn[i]==0) dfs( i);


    for(int i=0;i<=N;i++)
        for(int j : g[i])
            if(id[i]!=id[j]) dag[id[i]].push_back(id[j]);

    for(int i=0;i<N;i++)

}
```

&lt;M二&gt;

每棟房子會有兩種狀況 : 是否可以走到噴射裝置，因此可以分成以下幾種狀況，

case 1  : 起始位置可以走到噴射裝置

那你可以走過所有可以走到噴射裝置的建築，就是走一段然後使用噴射裝置跳到其它地方。

最後一次使用噴射裝置會跳到一個不可以走到噴射裝置，只要從中選一條最長的路走即可。

case 2 : 起始位置不可以走到噴射裝置

計算當前可以走最多的不同建築數，要特別注意左、右方有高度一致的狀況，也就是可以往某個方向走一段再折返，這點在 case 1 易知會被其它可能包含住，所以不用特別計算。


Time : $O(N)$

<br>

實作上，兩個陣列 $L$ , $G$ 分別代表向兩側延伸是非嚴格遞減、非嚴格遞增的狀況，避免使用 `while` 退化到 $O(n^2)$。

Code : 
```cpp
#include<bits/stdc++.h>
using namespace std;

#define p pair<int,int>
#define F first
#define S second

int main(){

    ios::sync_with_stdio(0), cin.tie(0);
    
    int N,K;
    cin>>N>>K; K--;
    
    vector<int>h(N) , x(N , 0);
    
    for(int i=0;i<N;i++) cin>>h[i];
    
    vector<p>G(N),L(N);
    for(int i=0;i<N;i++){
        L[i].F = 1; G[i].F = i;      
        if(i && h[i-1] <= h[i]) L[i].F = L[i-1].F + 1;
        if(i && h[i-1] >= h[i]) G[i].F = G[i-1].F;
    }

    for(int i=N-1;i>=0;i--){
        L[i].S = 1; G[i].S = i;
        if(i+1 < N && h[i] >= h[i+1]) L[i].S = L[i+1].S + 1;
        if(i+1 < N && h[i] <= h[i+1]) G[i].S = G[i+1].S;
    }

    char ch;
    for(int i=0;i<N;i++){
        cin>>ch;
        if(ch=='T'){
            x[G[i].F]++;
            if(G[i].S + 1 < N) x[G[i].S + 1]--;
        }
    }

    for(int i=1;i<N;i++) x[i] += x[i-1];

    if(x[K]){
        int res = 0,nw = 0;
        for(int i=0;i<N;i++){
            if(x[i]) res++;
            else nw = max({L[i].F , L[i].S , nw});
        }
        cout<<res + nw;
    }else{
        int l = K-1 , r = K+1;
        while(l>=0 && h[l]==h[K]) l--;
        while(r<N && h[r]==h[K]) r++;
        cout<<max(L[l+1].S , L[r-1].F);
    }
    return(0);
}
```
