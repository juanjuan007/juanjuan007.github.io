### 1868-噴射裝置

[IOJ 1868](https://tioj.ck.tp.edu.tw/problems/1868)

<br>

**題敘**

有 $n$ 棟房子，每棟建築有高度 $h_{i}$。

一開始你在第 $k$ 棟，每次你可以往相鄰且高度不增的建築移動，

其中某些建築物上會有噴射裝置，若你到那幾棟則可以無視上述規則，走到任意一棟。

你一開始在第 $k$ 棟，求最多可以走過多少不同建築。

$1 \leq n \leq 3 \times 10^{5}$ 、 $1 \leq k \leq n$ 、 $1 \leq h_{i} \leq 10^{6}$

<br>


**思路**

<br>

&lt;M一&gt;

將可從 $a$ 走到 $b$，視為存在一條有向邊 $a \rightarrow b$；有噴射裝置，代表對於每個節點都有一條邊。

至此，只要 SCC 縮點 + DAG DP 就可以解決問題。

Time : $O(n^{2})$ , TLE 其實是先 MLE 了。

<br>

Code :

```cpp
#include<bits/stdc++.h>
using namespace std;

int n , k;

vector<vector<int>>g , rg , dag;
vector<int>scc ,sz, ord , vis , dp , tag; 

void init(){
    g.resize(n);
    rg.resize(n);
    tag.resize(n);
    dag.resize(n);
    scc.resize(n);
    sz.resize(n);
    dp.resize(n);
}

void dfs1(int v){
    vis[v] = 1;
    for(int u : g[v]){
        if(vis[u]) continue;
        dfs1(u);
    }
    ord.push_back(v);
}

void dfs2(int v , int id){
    vis[v] = 1;
    scc[v] = id; sz[id]++;
    for(int u : rg[v]){
        if(vis[u]) continue;
        dfs2(u , id);
    }
}

int dfs3(int v){
    if(dp[v]) return(dp[v]);

    dp[v] = sz[v];
    
    for(int u : dag[v]){
        dp[v] = max(sz[v] + dfs3(u) , dp[v]);
    }
    return(dp[v]);
}

void kosaraju(){
    vis.assign(n,0);
    for(int i=0;i<n;i++)
        if(vis[i]==0) dfs1(i);

    reverse(ord.begin(),ord.end());

    vis.assign(n,0);
    int id = 0;
    for(int i : ord)
        if(vis[i]==0) dfs2(i ,id++);

    for(int i=0;i<n;i++){
        for(int j : g[i]){
            if(scc[i]==scc[j]) continue;
            dag[scc[i]].push_back(scc[j]);
        }
    }

    cout<<dfs3(scc[k]);
}



int main(){

    ios::sync_with_stdio(0) , cin.tie(0);

    cin>>n>>k; k--;

    init();
    
    vector<int>h(n);
    for(int i=0;i<n;i++) cin>>h[i];

    for(int i=0;i<n;i++){
        if(i && h[i] >= h[i-1]){ g[i].push_back(i-1); rg[i-1].push_back(i); }
        if(i < n-1 && h[i] >= h[i+1]){ g[i].push_back(i+1);  rg[i+1].push_back(i); }
    }

    char ch;
    for(int i=0;i<n;i++){
        cin>>ch;
        if(ch=='.') continue;

        g[i].resize(n-1);
        for(int j=0;j<n;j++){
            if(i==j) continue;
            g[i].push_back(j);
            rg[j].push_back(i);
        }
    }

    kosaraju();
    return(0);
}
```

&lt;M二&gt;

每棟房子會有兩種狀況 : 是否可以走到噴射裝置，因此可以分成以下幾種狀況，

case 1  : 起始位置可以走到噴射裝置

那你可以走過所有可以走到噴射裝置的建築，就是走一段然後使用噴射裝置跳到其它地方。

最後一次使用噴射裝置會跳到一個不可以走到噴射裝置，只要從中選一條最長的路走即可。

case 2 : 起始位置不可以走到噴射裝置

計算當前可以走最多的不同建築數，要特別注意左、右方有高度一致的狀況，這點在 case 1 易知會被其它可能包含住，所以不用特別計算。

Time : $O(n)$

<br>

實作上，兩個陣列 $L$ , $G$ 分別代表向兩側延伸是非嚴格遞減、非嚴格遞增的狀況，避免使用 `while` 退化到 $O(n^2)$。

Code : 
```cpp
#include<bits/stdc++.h>
using namespace std;

#define p pair<int,int>
#define F first
#define S second

int main(){

    ios::sync_with_stdio(0), cin.tie(0);
    
    int n,k;
    cin>>n>>k; k--;
    
    vector<int>h(n) , x(n , 0);
    
    for(int i=0;i<n;i++) cin>>h[i];
    
    vector<p>G(n),L(n);
    for(int i=0;i<n;i++){
        L[i].F = 1; G[i].F = i;      
        if(i && h[i-1] <= h[i]) L[i].F = L[i-1].F + 1;
        if(i && h[i-1] >= h[i]) G[i].F = G[i-1].F;
    }

    for(int i=n-1;i>=0;i--){
        L[i].S = 1; G[i].S = i;
        if(i+1 < n && h[i] >= h[i+1]) L[i].S = L[i+1].S + 1;
        if(i+1 < n && h[i] <= h[i+1]) G[i].S = G[i+1].S;
    }

    char ch;
    for(int i=0;i<n;i++){
        cin>>ch;
        if(ch=='T'){
            x[G[i].F]++;
            if(G[i].S + 1 < n) x[G[i].S + 1]--;
        }
    }

    for(int i=1;i<n;i++) x[i] += x[i-1];

    if(x[k]){
        int res = 0,nw = 0;
        for(int i=0;i<n;i++){
            if(x[i]) res++;
            else nw = max({L[i].F , L[i].S , nw});
        }
        cout<<res + nw;
    }else{
        int l = k-1 , r = k+1;
        while(l>=0 && h[l]==h[k]) l--;
        while(r<n && h[r]==h[k]) r++;
        cout<<max(L[l+1].S , L[r-1].F);
    }
    return(0);
}
```
